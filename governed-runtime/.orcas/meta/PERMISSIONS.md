# 权限

| 对象或状态 | 自动新建 | 自动修改 | 自动删除 | 要求 |
|---|---|---|---|---|
| `inbox` | 是 | 是 | 否 | 仅作为待处理输入 |
| `sources` | 是 | 否 | 否 | 文件名冲突时创建新版本 |
| `draft` | 是 | 是 | 否 | 记录来源与更新时间 |
| `trusted` | 否 | 否 | 否 | 修改必须创建确认项；旧 `verified` 兼容读取 |
| `stale` | 否 | 否 | 否 | 复核后由人决定状态 |
| `disputed` | 否 | 否 | 否 | 必须保留冲突来源 |
| `superseded` | 否 | 否 | 否 | 只读追溯 |
| `rule` | 否 | 否 | 否 | 达到证据门槛并明确批准 |
| `work/_publish/Published` | 否 | 否 | 否 | 必须明确批准对外发布 |
