# Metrics

每周记录一次，避免为了指标增加实时埋点。

| 周期 | 任务数 | 人工分类次数 | 确认项数 | 错误知识引用 | 知识复用次数 | 维护分钟 | 节省分钟 |
|---|---:|---:|---:|---:|---:|---:|---:|
| YYYY-Www | 0 | 0 | 0 | 0 | 0 | 0 | 0 |

## 计算

- 确认项比例 = 确认项数 / 任务数
- 净节省时间 = 节省分钟 - 维护分钟
- 维护占比 = 维护分钟 / 节省分钟
- 上下文命中率需在任务复盘中记录“实际使用的上下文条目 / 注入条目”

不使用知识页数量作为成功指标。
